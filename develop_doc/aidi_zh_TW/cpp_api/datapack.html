class DataPack#

DataPack handles the operations of a data package.

Public Functions

explicit DataPack(const std::string &path, bool create_if_not_exist = false, bool read_only = false)#

Open a data package at the given path. If create_if_not_exist is set, will create a new data package if the file does not exist.

Parameters:

  • path – path to the data package. Must ends with “.vfpack”.

  • create_if_not_exist – if set, will create a new data package when file does not exist, otherwise exception will be thrown

  • read_only – if set, datapack will be opened in read-only mode. All modifications to a read-only datapack throws. Closing a read-only datapack will be faster.

Throws:

  • excepts::InvalidParameter – if path does not end with “.vfpack”

  • excepts::FileNotFound – if path points to a non-existing file

  • excepts::CanNotOpenFile – if cannot open file for writing

  • excepts::DataPackInvalid – if the file is not a valid data package

void close(util::IProgressCallback *progress_callback = nullptr)#

close the data package, store all changes to disk

Attention

after close, all actions on this data package will throw

Parameters:

progress_callback – a callback to report the progress of the operation, nullptr if not needed

const std::vector<float> &transform3d_coefficient() const#

Get the coefficient of mapping from 2D image pixels to 3D points, which are stored in the vector are in the order (x_scale, y_scale, z_scale, x_offset, y_offset, z_offset). Note that if the returned vector of float is empty, the corresponding visionflow::img::Image of visionflow::datapack::ImageInfo represents a 2D image.

Returns:

const std::vector<float> transform3d_coefficient of the image.

ImageInfo &set_transform3d_coefficient(std::vector<float> transform3d_coefficient)#

Set the coefficient of mapping from 2D image pixels to 3D points.

Parameters:

transform3d_coefficient – New transform 3D coefficient to set, the vector should contain 6 float values (x_scale, y_scale, z_scale, x_offset, y_offset, z_offset), with the absolute values of the index at 0~2 should be greater than FLT_EPSILON or be empty (in this case the visionflow::img::Image represents a 2D image).

Throws:

excepts::InvalidArgument – The size of the argument transform3d_coefficient is not 6 or 0, or the absolute value of any of transform3d_coefficient with index at 0~2 (x_scale, y_scale, z_scale) is less than or equal to FLT_EPSILON if the coefficient contains 6 values.

Returns:

ImageInfo& reference to this object, for chaining.
