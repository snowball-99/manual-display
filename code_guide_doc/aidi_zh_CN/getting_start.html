脚本编辑器入门指南#

函数模板#

函数模板功能说明#

AIDI脚本编辑器提供了一些常用的辅助函数，可以直接作为函数模板插入脚本中。同时提供了函数模板管理功能，以便用户可以将常用的函数封装为模板在后续编辑脚本时复用。内置函数模板请参考 内置函数模板说明

如何查看已有函数模板#

单击右上角“函数模板管理”按钮，即可打开函数管理模板界面，其中左侧列表中显示的便是当前已有的模板函数列表。

自定义函数模板#

  1. 脚本编辑器-右上角函数模板管理

  2. 新增-输入函数别名-输入函数描述-输入编辑好的函数内容-点击注册

  3. 注册检查：函数无重名，无单函数，和其他未知不可运行原因。如不符合要求需进行修改后注册(如果希望暂时通过注册检查到下一步，可Ctrl+Z恢复到初始状态，或者注释掉所有当前函数内容)

批量导出-批量导入函数模板#

AIDI支持将函数模板批量导出然后在其他机器上导入以实现函数模板的多人共享。

  1. 脚本编辑器-右上角函数模板管理

  2. 选择“函数列表”旁边的函数导出标识-选择导出路径-完成批量导出

  3. 在需要导入的AIDI任意工程的综合判定节点，选择“函数列表”旁边的函数导入标识-批量选择函数所在地址-完成批量导入（函数导入会进行函数别名、函数名称去重）-完成批量导入

如何将函数模板插入脚本编辑区#

在脚本编辑区选定所需插入函数模板的代码行，输入 templ. ，脚本编辑器会列出所有的函数模板，然后选中需要插入的函数模板即可直接插入。

用户变量#

用户变量功能说明#

AIDI模型导出后，为了方便二次开发时对脚本变量的修改，建议用户将所有可能需要设置的变量均使用用户变量进行管理而不是直接写入脚本代码中。当前用户变量仅支持Number，String，Bool类型。

如何设置用户变量#

在左侧用户变量设置区单击添加用户变量，并设值即可。

如何在脚本中使用用户变量#

用户变量均存储在全局变量 VFLOW_USER_VARS 中， VFLOW_USER_VARS 是一个python dict。假设要访问名称为var0的用户变量，可以直接在脚本中使用 VFLOW_USER_VARS["var0"] 进行访问。

便捷调试#

AIDI脚本支持直接将变量通过 print 函数打印到日志区，并可直接在左下角脚本日志区查看。
